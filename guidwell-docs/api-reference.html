Embedding Guide

Embed Your Wizard

Once your wizard is set up, adding it to any page takes just a few clicks. There are three ways to do it — pick the one that matches how you build your site.

ℹ️
The Guidwell wizard is self-contained — it doesn't inherit your theme's styles, so it looks consistent on any theme. If you'd like it to blend in with your site's colors, turn on Use theme colors in the Settings tab.
⚠️
One wizard per site. Guidwell currently supports one active wizard at a time. Support for multiple wizards is coming in a future update.

Method 1 — Shortcode

The shortcode is the simplest option. It works in the Classic Editor, Elementor HTML widgets, Divi code modules, and anywhere else that accepts WordPress shortcodes.

Just paste this into your page or post:

[guidwell]
How to add it in the Classic Editor
  1. Open or create the page you want to embed the wizard on.
  2. Click in the body of the page where you want the wizard to appear.
  3. Type or paste [guidwell].
  4. Publish or update the page. The wizard will appear in its place on the front end.
✅
If your page builder has an HTML block or shortcode widget, you can use the same [guidwell] shortcode there too.

Method 2 — Gutenberg Block

If you use WordPress's built-in block editor, Guidwell adds its own block that you can insert just like any other block.

How to add the Gutenberg block
  1. Open or create the page in the block editor.
  2. Click the + button to add a new block (either in the toolbar or inline between blocks).
  3. Search for Guidwell in the block search bar.
  4. Click the Guidwell Wizard block to insert it.
  5. Drag it to the position you want on the page, then publish or update.

Method 3 — Elementor Widget

If you build with Elementor, Guidwell registers a native widget you can drag directly into your layout.

How to add the Elementor widget
  1. Open your page in the Elementor editor.
  2. In the left panel, use the search bar at the top to search for Guidwell Wizard.
  3. Drag the widget from the panel onto your layout wherever you'd like the wizard to appear.
  4. Click Publish or Update to save.
✅
You can place the Guidwell widget inside any Elementor section or column, including full-width layouts.

Where to Embed It

Not sure which page to put the wizard on? Here are the most common placements that work well:

Dedicated Landing Page
Create a page like "Find the Right Plan" or "Get a Recommendation" and make the wizard the main feature. Great for driving traffic from ads or email.
Homepage Section
Embed the wizard below your hero section so visitors can self-select right away. Works especially well for service-based businesses.
Pricing Page
Help visitors who are comparing plans find the right fit before they bounce. The wizard sits naturally alongside a pricing table.
Contact or Inquiry Page
Let visitors self-qualify before reaching out. You'll get better-matched inquiries and they'll feel more confident about what they need.
