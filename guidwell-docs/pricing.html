Billing & License

Annual subscription

Guidwell is licensed annually. Your license key activates the plugin on up to the allowed number of WordPress sites. Renewal happens once per year — if you don't renew, the plugin continues to work but automatic updates stop.

License keys

After purchase you receive a license key by email. Enter it under WP Admin → Guidwell → Settings → License. The license is validated against your domain.

What happens when a license expires?

The wizard on the frontend keeps running — visitors still see the wizard and get results. What stops: automatic plugin updates. You can renew at any time to re-enable updates.

Refunds

We offer a 14-day money-back guarantee. Contact support within 14 days of purchase if Guidwell doesn't work for your use case.

ℹ️
Guidwell uses GitHub Releases for automatic updates. When a new version is tagged, WordPress's update system will prompt you to update from inside WP Admin — no manual zip upload needed.

Frequently Asked Questions

Can I use one license on multiple sites? ⌄
Starter is limited to 1 site, Pro to 5 sites, and Agency to 25 sites. If you need more sites than your current plan allows, upgrade and the same license key will activate on additional installs.
Does the wizard slow down my site? ⌄
No. The wizard script is loaded asynchronously with defer and the config is a single REST API call cached by the browser. It doesn't block page rendering. The JavaScript bundle is under 30kb gzipped.
What happens to visitor data? ⌄
Guidwell doesn't persist visitor answers or results in the database — only the wizard configuration is stored. If you enable email capture, the visitor's name and email are included in your admin notification email but not stored in WP. Your WordPress site is the only server involved.
Can I switch from Linear to Tree mode later? ⌄
Yes. You can switch between modes at any time from the Logic tab. Switching to Tree mode preserves your questions. Switching back to Linear from Tree mode removes all branch connections — you'll be warned before that happens.
How do I upgrade from Starter to Pro or Agency? ⌄
Purchase the new tier from the Guidwell website. You'll receive a new license key. Go to WP Admin → Guidwell → Settings → License, replace the old key with the new one, and save. The tier upgrades immediately.
Does it work with any WordPress theme? ⌄
Yes. The wizard is self-contained in a shadow-like scope — it doesn't inherit theme CSS. The only thing that touches your theme is the shortcode container div, which you can style with standard CSS. The "Use theme colors" option reads CSS custom properties from your theme if they're available.
Is there a free tier or free trial? ⌄
Yes — Guidwell has a free tier. It's limited to 5 questions per wizard and linear mode only (no branching logic). It's intended for evaluation before purchasing a paid plan. Upgrade at any time to unlock all features.
Will my wizard break if I don't renew? ⌄
No. The wizard on the frontend will keep working indefinitely. The only thing that stops without a valid license is the WordPress admin update prompt — you won't receive automatic updates until you renew.
